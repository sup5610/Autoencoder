LOSSY COMPRESSION
Uses autoencoders to compress images in a lossy fashion.

- Modules used & version: matplotlib 3.8.4, numpy 1.26.4, torch 2.6.0+cu118, time,
                IPython 8.23.0, cv2 (opencv-python 4.11.0.86)

- CUDA optional

- Files: subset_1.npy, subset_2.npy, subset_3.npy, encoder_parameters.pth, decoder_parameters.pth
need to be in the same folder as Autoencoder.ipynb.
